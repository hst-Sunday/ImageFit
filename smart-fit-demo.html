🧠 智能缩放模式演示

🎯 智能默认策略 SMART

系统现在会根据输入参数智能选择最佳的缩放模式：

📏 单维度缩放

width: 800, height: undefined ↓ 默认使用: fit = 'inside' 保持原图比例，适应指定宽度

适用场景：等比例缩放，保持图像完整性

📐 双维度缩放

width: 800, height: 600 ↓ 默认使用: fit = 'cover' 保持比例，填充目标区域

适用场景：生成缩略图，填充指定区域

🎨 灵活覆盖：可通过 fit 参数显式指定模式，覆盖智能默认行为

📋 智能策略详解

输入条件 智能选择 行为说明 典型用途
width: 800, height: null inside 保持比例，宽度=800，高度自适应 响应式图片、等比例缩放
width: null, height: 600 inside 保持比例，高度=600，宽度自适应 纵向布局、高度限制
width: 800, height: 600 cover 保持比例，填充800x600区域，可能裁剪 头像、卡片缩略图
任何条件 + fit: 'fill' fill 强制精确尺寸，可能变形 固定尺寸要求

💡 使用建议

🔧 API 调用示例

# 智能单维度缩放 (自动使用 inside) curl -X POST http://localhost:8000/api/resize \ -F "image=@photo.jpg" \ -F "width=800" # 智能双维度缩放 (自动使用 cover) curl -X POST http://localhost:8000/api/resize \ -F "image=@photo.jpg" \ -F "width=800" \ -F "height=600" # 显式指定模式 (覆盖智能选择) curl -X POST http://localhost:8000/api/resize \ -F "image=@photo.jpg" \ -F "width=800" \ -F "height=600" \ -F "fit=fill"

🎨 最佳实践

  • 响应式图片：只设置宽度，让高度自适应
  • 卡片缩略图：设置宽高，使用默认 cover 模式
  • 完整显示：设置宽高 + fit=contain
  • 精确尺寸：设置宽高 + fit=fill
